# X Unlike Script

Simple JavaScript script to automatically remove likes from X (Twitter).

## How to Use

1. Open your likes page:
   https://x.com/yourusername/likes

2. Press **F12** to open Developer Tools.

3. Go to the **Console** tab.

4. Paste the script and press **Enter**.

The script will automatically remove likes one by one.

## Note

This script works slowly to reduce rate limits from X.
